## Commands

1. `daml new NumbersDemo --template empty-skeleton`
2. `cd .\NumbersDemo\`
3. `daml studio`
4. App Intro:
   - It will explain Numbers Data Type in Daml
5. Create a file Main.daml inside daml folder and add template code
